# hh99-ReactPlus
###### 항해 99 3기 React 심화반 강의 실습 작업물
###### 제공된 사이트 와이어프레임을 토대로 만들기

## 사이트에 필요한 기능
- 사용자별 계정 (로그인, 회원가입, 로그아웃 등)
- 이미지와 코멘트가 포함된 게시글 올리기
- 타인의 게시글에 댓글 달기 
- 댓글이 달리면 알림 받기
- 내 정보에서 닉네임 변경하기

## 진행도
### 페이지 뷰 (이후 디자인적 개선 필요)
- 로그인
- 회원가입
- 메인
- 게시글 상세
- 게시글 작성 / 수정


### 기능
- 사용자별 계정
